# ⚡ Key Counter - 키 연타 횟수 측정기

실시간 키 연타 횟수 측정 프로그램입니다.

## ✨ 주요 기능

- 🎯 **연타 측정**: 키를 누르고 뗄 때마다 정확히 1회씩 카운트
- 🌐 **전역 키보드 후킹**: 다른 프로그램 실행 중에도 백그라운드에서 작동 (관리자 권한 실행 필요)
- ⚡ **실시간 표시**: 키를 누르는 즉시 화면에 카운트 반영
- 🔄 **제어**: 시작, 정지, 초기화 기능 제공

## 🚀 설치 방법

### 1. Python 설치 확인

```bash
python --version
```

Python이 설치되어 있지 않다면 다운로드하세요.

### 2. 필수 라이브러리 설치

**중요**: 다른 `keyboard` 라이브러리와 충돌을 방지하기 위해 먼저 제거합니다.

```bash
pip uninstall keyboard
pip install pynput
```

### 3. 프로그램 실행

```bash
python key_counter.py
```

## 📖 사용 방법

### 1️⃣ 키 선택
- **"키를 선택하려면 클릭하세요"** 버튼 클릭
- 측정하고 싶은 키를 누르세요 (예: Space, A, Left Ctrl 등)
- 선택된 키가 표시됩니다

### 2️⃣ 측정 시작
- **"▶ 시작"** 버튼 클릭
- 선택한 키를 눌렀다 뗄 때마다 카운트가 증가합니다
- 다른 프로그램을 실행해도 계속 측정됩니다

### 3️⃣ 측정 정지
- **"⏸ 정지"** 버튼 클릭
- 카운트 측정이 일시 정지됩니다
- 다시 시작 버튼을 누르면 이어서 측정 가능합니다

### 4️⃣ 카운트 초기화
- **"초기화"** 버튼 클릭
- 카운트가 0으로 리셋됩니다

## 🎮 사용 목적

- **게임**: 게임 플레이 중 특정 키(예: 스페이스바) 연타 횟수 측정
- **타이핑 연습**: 특정 키 타이핑 횟수 카운트
- **작업 분석**: 특정 키 사용 빈도 분석

### 버튼이 화면에 보이지 않음

모니터 해상도가 낮은 경우 창 크기를 조정할 수 있습니다:

```python
# key_counter.py 10번째 줄

```

## 📝 기술 스택

- **언어**: Python 3.7+
- **GUI**: tkinter (Python 기본 내장)
- **키보드 후킹**: pynput
- **스레딩**: threading (백그라운드 작동)

## 🔒 개인정보 보호
- 선택한 키의 연타 횟수만 측정합니다
- 입력된 텍스트나 다른 키 입력은 기록하지 않습니다
- 인터넷 연결이 필요하지 않습니다
- 데이터를 외부로 전송하지 않습니다
